# SG15 norm-conserving pseudopotentials

This directory contains SG15 ONCV pseudopotential files (`*.psp8`) used by SPARC-atomSFE for **rSCAN** calculations.

## Source

The potentials were taken from the SG15 ONCV library maintained at:

- http://www.quantum-simulation.org/potentials/sg15_oncv/index.htm

## Reference

Please cite the following publication when using these potentials:

M. Schlipf and F. Gygi, *Computer Physics Communications* **196**, 36 (2015).  
https://doi.org/10.1016/j.cpc.2015.05.011
