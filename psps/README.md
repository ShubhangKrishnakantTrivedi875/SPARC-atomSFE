## SPMS table of pseudopotentials
* See https://github.com/SPARC-X/SPMS-psps for details

**Which folder to use:** For **GGA_PBE** and **PBE0**, use the pseudopotentials in this directory (`psps/`). For **LDA** functionals, use `psps/SPMS/`. For **SCAN** functionals, use `psps/SG15/`. Set `psp_dir_path` accordingly when running pseudopotential calculations.
